# 创作者社区 - 管理后台

这是一个基于 React + TypeScript + Vite 构建的创作者社区管理后台系统。

## 项目特点

- **审核流管理**：包含创作者审核、作品审核等完整流程。
- **创作者画像**：详尽的创作者详情页，包含数据统计、作品列表、勋章管理等。
- **精美设计**：基于 Ant Design 进行二次开发，拥有现代感十足的 UI 设计和流畅的交互体验。
- **高性能**：采用现代化的技术栈，确保页面的极速加载和响应。

## 技术栈

- **前端框架**: React 18
- **构建工具**: Vite
- **UI 组件库**: Ant Design
- **路由管理**: React Router 6
- **样式处理**: Vanilla CSS

## 开发指南

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产环境
```bash
npm run build
```
